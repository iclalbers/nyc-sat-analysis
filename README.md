# NYC SAT Analysis

This project analyzes NYC public school SAT performance using Python and pandas.

## Questions Answered

- Which NYC schools have the best math results?
- What are the top 10 performing schools based on combined SAT scores?
- Which borough has the largest standard deviation in the combined SAT score?

## Files

- `main.py` → main Python analysis code
- `schools.csv` → SAT dataset
- `README.md` → project description

## Technologies Used

- Python
- pandas

## How to Run

Run the following command:

```bash
python main.py
```

## Output

The program prints:

- best math schools
- top 10 SAT schools
- borough with highest SAT score standard deviation
